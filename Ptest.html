#标签 问题 答案
signature/加密/解密 signature字段，我是用公钥加密它，还是用私钥加密它？ signature加密用私钥，解密用公钥。
api/參數 在queryIncreaseCreditRecord这支api中, reqNo与queryReqNo的差异 reqNo 是 ebet 产生, 用来记录每一笔我们对贵方的 api 请求. queryReqNo 是记录在贵方, 让eBet查询贵方对 increseCredit 事件处理状况所依据的 reqNo.
api/參數 在文件中各参数对大小写与特殊符号有限制吗吗? 有限制,需要与文件中规范的相同,否则会造成错误.
refundseqno/退款 refundseqno是来检查哪笔下注资料来退款呢？ refundseqno是对应increaseCredit事件中 betlist的seqno
increaseCredit/下注 怎么得知玩家的下注资料? 透过API：increaseCredit 当玩家下注时我们会立即发送请求通知
白名单 白名单有哪几个部分? 白名单需要提供IP做添加。如果没有特别要求，则预设皆为开放。白名单有三个部分，游戏前端、呼叫我方ebetAPI、数据后台登入。
token 如何直接跳转进入游戏大厅? 可以使用此URL:http://<'provide by ebet'>/?infoUrl=h5/<'provide by ebet'>&username=test&accessToken=abc12345，就可以直接进入游戏大厅。
api/參數 seqNo的值是唯一的吗? 在increaseCredit这个API，seqNo的值是唯一的。
多钱包 多钱包主要有几种登入方式? 多钱包分为三种登入方式，普通登入，Token登入，AppLinks登入。
api/參數 如果我想要利用 updatebetlimit 设定一个玩家的投注限额, 我该如何设定参数? ebet 利用游戏类型来管理玩家的投注限额, 如果要更改一个游戏的投注限额, 就必须要设定这个游戏类型中的所有参数.
