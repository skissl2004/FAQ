#標籤 問題 答案
signature/加密/解密 signature字段，我是用公鑰加密它，還是用私鑰加密它？ signature加密用私鑰，解密用公鑰。
api/參數 在queryIncreaseCreditRecord這支api中, reqNo與queryReqNo的差異 reqNo 是 ebet 產生, 用來記錄每一筆我們對貴方的 api 請求. queryReqNo 是記錄在貴方, 讓eBet查詢貴方對 increseCredit 事件處理狀況所依據的 reqNo.
api/參數 在文件中各參數對大小寫與特殊符號有限制嗎嗎? 有限制,需要與文件中規範的相同,否則會造成錯誤.
refundseqno/退款 refundseqno是來檢查哪筆下註資料來退款呢？ refundseqno是對應increaseCredit事件中 betlist的seqno
increaseCredit/下註 怎麽得知玩家的下註資料? 透過API：increaseCredit 當玩家下註時我們會立即發送請求通知
白名單 白名單有哪幾個部分? 白名單需要提供IP做添加。如果沒有特別要求，則預設皆為開放。白名單有三個部分，遊戲前端、呼叫我方ebetAPI、數據後台登入。
token 如何直接跳轉進入遊戲大廳? 可以使用此URL:http://<'provide by ebet'>/?infoUrl=h5/<'provide by ebet'>&username=test&accessToken=abc12345，就可以直接進入遊戲大廳。
api/參數 seqNo的值是唯一的嗎? 在increaseCredit這個API，seqNo的值是唯一的。
多錢包 多錢包主要有幾種登入方式? 多錢包分為三種登入方式，普通登入，Token登入，AppLinks登入。
api/參數 如果我想要利用 updatebetlimit 設定一個玩家的投註限額, 我該如何設定參數? ebet 利用遊戲類型來管理玩家的投註限額, 如果要更改一個遊戲的投註限額, 就必須要設定這個遊戲類型中的所有參數.
